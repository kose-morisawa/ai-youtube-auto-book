## 動画1本に何時間かかるか、知っていますか？

YouTubeで収益化するには、最低でも**月4〜8本の投稿**が必要と言われています。

では、1本あたりの制作時間はどのくらいか。

| 作業 | 一般的な時間 |
|------|------------|
| リサーチ・企画 | 2〜3時間 |
| 台本作成 | 3〜5時間 |
| 撮影・録音 | 1〜3時間 |
| 編集 | 4〜8時間 |
| サムネイル | 1〜2時間 |
| **合計** | **11〜21時間** |

月8本なら、**90〜160時間**。副業でこれをこなすのは、現実的に不可能です。

## 「続かない」のは意志の問題じゃない

よく「続けることが大事」と言われますが、本質的な問題は別にあります。

**制作コストが高すぎる。**

1本作るたびに10時間以上かかるなら、仕事・学業・生活と並行して続けられる人はほぼいません。「意志が弱い」のではなく、**システムが間違っている**のです。

## AIと自動化で、ゲームのルールを変える

本書で紹介するシステムは、以下の工程をほぼ自動化します。

```
台本生成（AI）
    ↓
音声合成（VOICEVOX）← 自動
    ↓
動画レンダリング（Remotion）← 自動
    ↓
素材収集（Pexels/Pixabay等）← 自動
    ↓
YouTubeアップロード ← 自動
```

**人間がやること：企画とテーマを決めるだけ。**

実際、このシステムで動画1本あたりの制作時間は**30分〜1時間**まで短縮できます。

## このシステムで何が実現するか

### 投稿頻度が上がる

月4本 → 月16本以上が現実的になります。投稿頻度はチャンネル成長に直結します。

### 複利的に成長する

動画は資産です。100本あれば、100本が24時間働き続けます。
このシステムがなければ100本作るのに1,000時間かかります。
あれば**150時間以下**で達成できます。

### 「仕組みを作って稼ぐ」が実現する

YouTube広告収入 → チャンネル成長 → 情報商材・コミュニティ収益
この経路で収益を積み上げていく。

本書はその「仕組みを作る」部分の完全な実装ガイドです。
