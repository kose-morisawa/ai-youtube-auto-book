## 収益化の3ステージ

このシステムを使った収益化ロードマップです。

```
Stage 1: 広告収益（〜1,000人）
    ↓
Stage 2: 情報商材・コミュニティ（〜10,000人）
    ↓
Stage 3: 配当金でFIRE（元本5,000万円）
```

---

## Stage 1：まず1,000人を目指す

YouTube収益化の最初の壁は**1,000チャンネル登録者**です。
この段階では広告収益よりも「チャンネルを育てること」が目標です。

### 投稿戦略

| 指標 | 目標 |
|------|------|
| 投稿頻度 | 週2〜3本（このシステムなら現実的） |
| 最初の10本 | 以下のテーマで網羅する |
| SEO | ロングテールキーワードを狙う |

### 最初に作るべき10本のテーマ例（マネー×AI軸）

1. 「新NISAとは何か？5分で完全理解」
2. 「AIで株価を予測できるか試してみた」
3. 「月3万円の積立、30年後にいくらになる？」
4. 「インデックス投資vs個別株、AIに聞いてみた」
5. 「貯蓄だけの人が気づいていない損失」
6. 「配当金でFIREするには元本いくら必要？」
7. 「iDeCoとNISAどっちが得？AIが計算した」
8. 「日本株 vs 米国株、20年分のデータをAIが比較」
9. 「副業でFIREする最速ルート、AIが提案」
10. 「投資初心者がやりがちな5つの失敗」

### タイトルのCTR設計

再生数はタイトルとサムネイルで9割が決まります。

**高CTRタイトルの型（3パターン）:**

| 型 | 例 |
|----|-----|
| 損失回避型 | 「知らないと損する」「貯蓄だけの人が気づかない〇〇」 |
| 好奇心ギャップ型 | 「AIに聞いたら想定外の答えが返ってきた」 |
| 数字・変化型 | 「月3万→30年後に〇〇万円になる話」 |

## Stage 2：情報商材・コミュニティで収益を伸ばす

チャンネル登録者が1,000人を超えたら、より大きな収益源を作ります。

### 情報商材の例

| 商品 | 価格帯 | 制作期間 |
|------|--------|---------|
| 「AI投資分析ツールの作り方」Zenn Book | ¥3,000〜¥8,000 | 1〜2週間 |
| 「YouTube自動化完全ガイド」（本書のような） | ¥4,980〜¥9,980 | 2〜4週間 |
| 投資コミュニティ月額会員 | ¥980〜¥2,980/月 | 継続運営 |

:::message
**本書自体がこの戦略の実践例です。**
このYouTube自動化システムを使いながら、同時に「このシステムの作り方」を
Zenn Bookとして販売することで、二重の収益が生まれます。
:::

### コンテンツの再利用戦略

1本の動画を複数の収益源に展開します。

```

    ↓
  詳細版・深掘り版
    ↓

    ↓
  コミュニティ限定コンテンツ
    ↓

```

## Stage 3：配当金でFIREする

YouTube収益と情報商材収益を**配当金が出る資産**に投資します。

### FIREに必要な元本の計算

```
年間生活費 = 300万円（月25万円）と仮定
必要元本  = 300万円 ÷ 4% = 7,500万円

もし年間生活費が200万円なら：
必要元本  = 200万円 ÷ 4% = 5,000万円
```

※ 4%ルール（年利4%の配当・分配金）を前提とした計算

### このシステムで元本形成を加速する

| 月収入（例） | 月投資額 | 25年後の資産（年利5%複利） |
|------------|---------|------------------------|
| 月収10万（広告+情報商材） | 5万円 | 約2,900万円 |
| 月収30万 | 15万円 | 約8,700万円 |
| 月収50万 | 30万円 | 約1.7億円 |

**動画を量産できるシステムがあれば、収益化の速度が上がります。**  
このツールは「FIREへの時間を短縮する」ために設計されています。

## 実際の投稿サイクル（週次ルーティン）

**週合計：約2時間。月8本以上の投稿が現実的になります。**

## 検索流入を増やすSEO戦略

AIで生成する動画コンテンツはSEOと相性が良いです。

### キーワード戦略

```bash
# WikipediaとNewsAPIで関連キーワードを自動収集
python tools/api/finance.py wiki "配当金 FIRE"
python tools/api/finance.py news "積立NISA 口座開設"
```

### タグの最適化

動画アップロード時に関連タグを自動設定できます。

```bash
node tools/api/youtube.mjs upload \
  --title "..." \
  --tags "NISA,新NISA,積立投資,iDeCo,インデックス投資,ETF,配当金,FIRE,副業,資産形成"
```

タグは **最大500文字** まで設定できます。
