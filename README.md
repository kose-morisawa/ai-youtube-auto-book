# AI YouTube Auto — Zenn Book

Zenn書籍「AIがYouTube動画を全自動生成する仕組みを作った」の原稿リポジトリです。

## ローカルプレビュー

```bash
npm install
npm run preview
# http://localhost:8000 で確認
```

## 構成

```
books/
```
